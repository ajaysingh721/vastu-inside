# Vastu Video Carousel

## Video Files Required

Place your Vastu consultation videos in this directory with the following names:

1. **vastu-1.mp4** - First video showcasing Vastu principles
2. **vastu-2.mp4** - Second video demonstrating Vastu applications
3. **vastu-3.mp4** - Third video showing client testimonials or results

### Video Specifications:

- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Aspect Ratio**: 16:9
- **Duration**: 15-30 seconds per video (for smooth carousel experience)
- **File Size**: Keep under 5MB per video for optimal loading

### Tips:

- Ensure videos are landscape orientation
- Use high-quality footage with good lighting
- Add captions or text overlays if needed
- Compress videos to balance quality and file size
- Test on mobile devices for performance
